An interactive visualization mapping 1,000 philosophers through semantic space. Each point is a thinker; its position reflects the intellectual distance between ideas — computed from 3,072-dimensional embedding vectors.

## How it works

For each of the 1,000 philosophers, **gpt-5.4** produced two texts:

- An **intellectual profile** — a dense description in the tradition's own technical vocabulary, covering ontology, epistemology, ethics, and main theses.
- A **philosophical stance** — a period-agnostic, cross-tradition analysis that identifies structural affinities with thinkers from other schools, regardless of historical context.

These texts, combined with key concepts extracted for each philosopher, were embedded into high-dimensional vectors using **OpenAI's text-embedding-3-large** (3,072 dimensions).

**UMAP** then projects those dimensions onto a 2D plane while preserving local structure: philosophers with similar ideas appear clustered together. The KNN network built on these vectors achieves **Precision@K=5 of 0.827** — over 8 in 10 nearest neighbors share the same historical period or school of thought.

## Navigation

| Action | Effect |
|--------|--------|
| Left-drag | Rotate the terrain (3D) / pan (2D) |
| Scroll wheel | Zoom in & out |
| Click a node | Open the philosopher's full profile |
| Search bar | Locate a specific thinker by name |

## Settings

The gear icon exposes several parameters:

- **Repulsion Strength** — controls how aggressively UMAP separates philosophical communities during optimization. Higher values push distinct traditions further apart.
- **Node appearance** — size scale, size contrast (prominence of major figures), and opacity.
- **Label visibility** — show labels for the top N philosophers by relevance, or all.
- **Terrain Mode** — toggle between a 3D density terrain and a flat 2D scatter plot. In 3D, denser philosophical communities form higher peaks. Terrain shape is fully configurable (resolution, bandwidth, height, perspective, vertical offset).

## Data

- **1,000 philosophers** ranked by Wikidata sitelinks (a proxy for cultural relevance)
- Profiles and stances generated by **gpt-5.4**
- Embeddings via **text-embedding-3-large** (OpenAI)
- Dimensionality reduction via **UMAP**

## Tech stack

Pure HTML + Canvas 2D — no external libraries. UMAP projections are pre-computed in Python and served as a single JSON file.
